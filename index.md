# Bruno.js

## What is bruno.js

Bruno.js is a fast, small JavaScript platform.

## Examples

Setting up strict mode:

```js
bruno.strict = true;
bruno.each(["hello", "world"]); // This will throw an error
bruno.strict = false;
bruno.each(["another", "array"]) // This will not throw an error
```
